# PandasHelpers

small collection of helper funktions inspired by pandas and sklearn

implemented so far:
- get_dummies!
- abs_max_norm!
- train_test_split
- naive_impute(!)


### Requirements:
- DataFrames
- Statsbase

### Getting started:
- clone repository
- navigate to folder and run `] activate .`, `add DataFrames`, `add Statsbase` in Julia REPL
- import package with `using PandasHelpers`
